# SaaS Product Analytics Dashboard 📊

# Overview

This project focuses on analyzing user behavior, retention, churn, and revenue trends for a simulated SaaS product.
The goal was to approach the project like a real Product Analyst by not only building dashboards, but also identifying business problems, extracting insights, and recommending product improvements based on data.


# Problem Statement
Many SaaS companies struggle with:
- High churn rates
- Low feature engagement
- Poor Free-to-Premium conversion
- Inconsistent user retention

This project was created to analyze these challenges and identify opportunities to improve user engagement and monetization.


# Objectives
- Analyze user growth and engagement trends
- Measure churn and retention patterns
- Understand feature usage behavior
- Evaluate subscription plan performance
- Generate product and business recommendations


# Tools Used
- SQL
- Excel
- Power BI
- DAX


# Dataset Information
The dataset contains simulated SaaS user data including:
- User signups
- Subscription plans
- Session activity
- Feature usage
- Revenue metrics
- Retention behavior
- Churn indicators

  
# Dashboard Features
1. Executive Overview
- Total Users
- Active Users
- Churn Rate
- Revenue Tracking
- Retention KPIs

2. User Behavior Analysis
- Session trends
- Feature engagement
- User activity analysis

3. Retention & Churn Analysis
- Churn distribution
- Retention comparison across plans
- Engagement impact on retention

4. Revenue & Conversion Analysis
- Revenue by subscription plan
- Free-to-Premium conversion funnel
- Monetization insights


# Key Insights
- Premium users generated the majority of revenue despite representing a smaller user segment.
- Free users showed the highest churn and lowest engagement levels.
- Users with higher session duration demonstrated better retention behavior.
- Features like Automation and Reports were associated with stronger engagement patterns.


# Recommendations
- Improve onboarding experience for Free users
- Promote high-retention features earlier
- Optimize conversion funnel strategies
- Focus more on high-quality acquisition channels
- Introduce re-engagement campaigns for inactive users
